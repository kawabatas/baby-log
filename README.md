# babylog
## リクログ（ベビーログ）
- 赤ちゃんのうんち、おしっこ、うんちとおしっこ、ミルクの時間をグーグルスプレッドシートに記録する
- 記録の方法は3種類

### 1.アレクサで記録
「アレクサ、リクログでうんち」
「アレクサ、リクログでおしっこ」
「アレクサ、リクログでうんちとおしっこ」
「アレクサ、リクログでミルク」
